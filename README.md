React Supabase Blog App
A multi-user blogging platform built using React and Supabase.
Users can sign up, log in, write blog posts, browse all blogs, and manage (edit/delete) their own posts with a simple, modern UI.

Features
User Authentication: Secure signup/signin with email using Supabase Authentication.

Home Page: Browse all published blogs from all users.

My Blogs: View, edit, or delete only your own posts on a dedicated "My Blogs" page.

Add Post: Create new blog posts from any page by clicking "Add Post" in the navbar.

Edit Post: Edit existing blogs, only visible for your own posts in "My Blogs".

Delete Post: Remove your own blogs, only from "My Blogs".

Detail Page: Click any blog to see its full content on a dedicated detail view.

Logout: Securely sign out and return to the login/signup view.


Tech Stack
React

Supabase (Authentication + Database)

React Router (Routing)
